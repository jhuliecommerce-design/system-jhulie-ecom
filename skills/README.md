# Skills

Skills são tarefas executáveis usadas pelos agentes.

Enquanto os agentes tomam decisões, as skills executam ações específicas.

## Como usar

1. Escolha um agente
2. Identifique a tarefa
3. Utilize a skill correspondente
4. Execute no Claude

## Estrutura

- Cada skill resolve um problema específico
- Deve ser simples, direta e reutilizável

## Conceito

Agente = decisão  
Skill = execução

Use ambos em conjunto para obter melhores resultados.
