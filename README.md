# AI Ecommerce Operating System

Você não está acessando prompts.

Você está acessando um sistema de operação com IA.

Cada agente representa um especialista da sua operação.

Use como um time.

---

## O que é isso

Este repositório é um sistema de agentes para ecommerce (dropshipping internacional).

Ele foi criado para:

* tomar decisões melhores
* executar mais rápido
* escalar operações com consistência

---

## Estrutura do sistema

```text
Claude (cérebro)
↓
Agentes (especialistas)
↓
Skills (execução)
↓
Integrações (dados e ação)
```

---

## O que você vai encontrar aqui

### agents/

Especialistas da operação

* Diretor → estratégia
* Gestor → análise
* Meta Ads → campanhas
* Google Ads → tráfego
* Produtos → validação
* SEO → estrutura
* SAC → atendimento
* Métricas → análise

---

### skills/

Execuções específicas

* análise de produto
* criação de criativos
* resposta de email
* estruturação de campanhas

## Como instalar (Terminal)

### 1. Instale o Git (se não tiver)

👉 https://git-scm.com

---

### 2. Clone o repositório

Abra o terminal e rode:

```bash
git clone https://github.com/jhuliecommerce-design/system-jhulie-ecom.git
```

---

### 3. Entre na pasta

```bash
cd system-jhulie-ecom
```

---

## Como usar

### Modo simples (recomendado)

1. Abra o Claude
2. Escolha um agente dentro da pasta `agents/`
3. Copie o conteúdo
4. Cole no Claude
5. Envie junto com seu contexto

---

### Exemplo

```text
Minha campanha está com ROAS 1.2, CPC alto e poucas compras
```

Use com o agente de Meta Ads.

---

### Modo avançado (opcional)

* usar VS Code
* usar terminal
* integrar com ferramentas externas
* evoluir para automação

---

## Regra mais importante

Agente bom + contexto ruim = resposta ruim
Agente bom + contexto bom = resultado forte
